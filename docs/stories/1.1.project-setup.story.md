# Story 1.1: Project Setup

## Status
Draft

## Story
**As a** developer,
**I want** to set up a new Spring Boot project with necessary dependencies (Web, Security, JPA, H2),
**so that** we have a foundational codebase to build upon.

## Acceptance Criteria
1. A new Spring Boot project is successfully created.
2. The `pom.xml` includes dependencies for Spring Web, Spring Security, Spring Data JPA, and the H2 database.
3. The application starts successfully without any errors.

## Tasks / Subtasks
- [ ] Initialize a new Spring Boot project using Spring Initializr or Maven archetype. (AC: 1)
- [ ] Add the following dependencies to `pom.xml`: (AC: 2)
  - `spring-boot-starter-web`
  - `spring-boot-starter-security`
  - `spring-boot-starter-data-jpa`
  - `h2` (runtime scope)
  - `jjwt-api`, `jjwt-impl`, `jjwt-jackson` for JWT support.
  - `spring-boot-starter-validation` for DTO validation.
- [ ] Create the main application class (`TaskFlowAccentureProjectsApplication.java`). (AC: 1)
- [ ] Configure the `application.properties` file for the H2 database (e.g., `spring.h2.console.enabled=true`). (AC: 2)
- [ ] Run the application to confirm it starts without errors. (AC: 3)
